# Enliven
# General info
A React Native job listing project with live updates on current job listings around the country.

# Prerequisities
```
node.js
expo
```


## To run this project, use Expo.
```
$ cd Enliven-master
$ expo start 
```
# Technologies
* Expo CLI 3.18.2
* React Native 0.62
